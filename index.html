Introduction

There is a crucial interrelation between microstructure and chemical, physical, and/or mechanical properties of materials.

In this course:

Characterization of the microstructure both qualitative and quantitative, as well as chemical composition.

The microstructural and chemical characterization relies mostly on the interaction of a material with a probe:

  • visible light
  • X-rays
  • beam of high energy electrons

The resultant signal will then be collected, processed and interpreted.

Visible light:

Microscopy: allows looking at objects not visible for the naked eye.

Limitation: the smallest distinguishable features are determined by the wavelength of visual light (0.4 - 0.7 μm)

Overview of various types of EM-radiation and their wavelength

Smaller wavelength than visible light => smaller details can be observed

Electron microscopes:

Electrons emitted from a heated filament are accelerated by a high voltage and then focused using magnetic lenses onto a sample, resolution in these devices is limited by lenses.

Electrons can be accelerated up to a velocity near the speed of light => electrons with a wavelength ≈ 0.1 nm => atomic and molecular scale of matter.

X-rays:

X-rays typically have a wavelength below 10 nm => in practice resolution is limited because of other reasons (focusing devices).

Their penetrating power is largely dependent on their energy: The energy of an X-ray is reversely proportional to its wavelength and is calculated by the equation E=hc/lambda

X-rays are not only for microscopic purposes (µm size), but in general provides a method for non-destructive imaging of objects of all sizes (up to meters).

Fast development in characterization techniques, digital revolution

=> massive amounts of digital data

up to the geologist to make a critical decision associated with:

  • sample selection
  • sample preparation
  • choice of characterization method
  • understanding and critical review of results

In this course we hope to clarify these steps, in order to prepare the geologist for a scientific way of interpreting digital data obtained by the techniques.

Some general terms:

  • Macrostructure: Those features visible to the naked eye. Ex: Large pores, big inclusions, cracks.
  • Mesostructure: those features that are on the borderline of the visible.
  • Microstructure: covers the scale of structural phenomena most commonly of concern to geologists, namely grain and particle size, microcracking and microporosity.
  • Nanostructure: restricted to sub-micrometre features including the width of grain boundaries. A nanostructure is an object of intermediate size between microscopic and molecular structures.

link between nano-, micro and macroscale processes inside porous material

What are "digital images"?

Digital images

Electronic snapshots taken of a scene or scanned from documents, sampled and mapped as a grid of dots or picture elements (pixels).

Pixels

Can anyone explain what a pixel is?

In digital imaging, a pixel, or picture element is a physical point in a raster image, or the smallest addressable element in an all points addressable display device, so it is the smallest controllable element of a picture represented on the screen.

Each pixel is assigned a tonal value (black, white, shades of gray or colour), which is represented in binary code. The binary digits ("bits") for each pixel are stored in a sequence by a computer and often reducedto a mathematical representation (compressed). The bits are then interpreted and read by the computer to produce an analog version for display or printing.

Pixel values: As shown in this bitonal image, each pixel is assigned a tonal value, in this example 0 for black and 1 for white.

Resolution

Resolution is the ability to distinguish fine spatial detail.

The spatial frequency at which a digital image is sampled (the sampling frequency) is often a good indicator of resolution.

Dots-per-inch (dpi) or pixels-per-inch (ppi) are common and synonymous terms used to express resolution for digital images.

the more pixels an image contains, the more it can be enlarged before the separate pixels start to show

Resolution in digital cameras

Very important parameter for the selection of the camera => Images must be able to record the fine detail revealed by the microscope magnification.

Often quoted as the size of charged coupled device (CCD), which is effectively the number of pixels on the chip. It should be noted: the size of the individual pixels varies amongst different types of CCDs.

Increasing resolution enables the capture of finer detail, but at some point added resolution will not result in an appreciable gain in image quality, only larger file size.

File size

    Determined by:

    bit depth

    number of pixels in the file

    1. = pixel dimensions
    2. = surface area of the document to be captured x dpi2

    Formula:

    1. File Size = (pixel dimensions x bit depth) / 8 [bytes]
    2. File Size = (height x width x dpi2 x bit depth) / 8 [bytes]

File size naming convention

Because digital images often result in very large files, the number of bytes is usually represented in increments of 210 (1,024) or more:

  • 1 byte = 8 bits
  • 1 Kilobyte [KB] = 1,024 bytes
  • 1 Megabyte [MB] = 1,024 KB
  • 1 Gigabyte [GB] = 1,024 MB
  • 1 Terabyte [TB] = 1,024 GB

Compression

= used to reduce image file size for

  • storage
  • processing
  • transmission

All compression techniques abbreviate the string of binary code in an uncompressed image to a form of mathematical shorthand, based on complex algorithms.

There are several compression techniques available.

Compression

2 kinds of compression:

  • Lossless schemes: abbreviate binary code without discarding any information
    => when image is "decompressed" it is bit for bit identical to original (bitonal images)
  • Lossy schemes: utilize a means for averaging or discarding the least significant information, based on an understanding of visual perception. However, it may be extremely difficult to detect the effects of lossy compression; (tonal images).

Compression

the effects of JPEG lossy compression: artifacts: pixel squares occur and fine details disappear (eyelashes)

File formats

TIFF:

  • Tagged Image File Format
  • lossless
  • The best file format for archiving high quality images
  • widely supported accross operating systems

File formats

JPG or JPEG

  • Joint Photographers Expert Group
  • lossy
  • are automatically compressed (level of compression can be chosen to desired image quality), resulting in a relatively small file size while still retaining quality
  • created for photographs (millions of colours) and ideal for email and Web use

File formats

GIF

  • Graphics Interchange Format
  • lossless
  • keeps edges and lines sharp: good for graphis
  • limited to 256 colours: not recommended for photographs

File formats

PNG

  • Portable Network Graphics format
  • lossless
  • open source alternative to GIF with higher compression rate
  • useful for image storage during editing

Image quality

What is considered a good quality image can vary:

  • Descriptions of a material: an image that closely resembles what one sees when looking in the microscope
  • analysis: may require enhancement of certain features, exaggeration of contrast, etc.
  • Multiple versions of the same image may be produced during preprocessing, during various phases of analysis, and as publications are prepared;
    the original uncompressed image must always be retained

image enhancement or preprocessing

often necessary to enhance features of interest in order to perform an analysis.
